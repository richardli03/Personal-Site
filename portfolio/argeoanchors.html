NOTE: This documentation file in my personal repository before we created the branch linked above contains some additional information about the project.


ARGeoAnchors are a technology created by Apple that allows a phone to be localized to a high degree of accuracy in outdoor settings. This is done using a combination of GPS + visual imagery of the surroundings. Apple has done something similar to Google and has physically mapped out huge swaths of space in a few specific locations. The combination of using GPS to get an initial location and visual imagery to fine-tune the location of the phone allows us to localize in outdoor settings with an error of roughly less than 1 meter.


The reason why this technology is so important to us is because one of the accessibility apps that OCCaM Lab is creating (Clew) relies heavily on an accurate localization to ensure its navigational errors are kept to a minimum. Previous solutions to this problem relied on visual matching algorithms, physically requesting that the user put their phone in the same place every time, etc.
